# TFTP
TFTP implementation in C (linux)

## Run
after compiling (file name : client.out, server.out)

### run server
```
server.out
```

### run client

```
client.out [hostname] [options]
```
options
 * `-s [filename]` send file to server
 * `-g [filename]` get file from server
 * `-n` `-o` netascii mode / octet mode
